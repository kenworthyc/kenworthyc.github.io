Iterators in Ruby and Javascript

ruby arrays

Note: these iterators are not destructive, and do not change the arrays directly.

.each

iterates through each element of an array, starting from the beginning.

arr = ['a', 'b', 'c', 1, 2, 3]
arr.each do |element|
print element * 2
end
# => aabbcc246

.reverse_each

accesses each element of an array, starting from the end.

arr = ['a', 'b', 'c', 1, 2, 3]
arr.reverse_each do |element|
print element
end
# => 321bca

.each_with_index

accesses each array element along its index.

arr = ['a', 'b', 'c', 1, 2, 3]
arr.each_with_index do |element, index|
print element, "/", index, " "
end
# => a/0 b/1 c/2 1/3 2/4 3/5

.each_index

accesses each index of an array (not its elements).

arr = ['a', 'b', 'c', 1, 2, 3]
arr.each_index do |index|
print index
end
# => 012345

ruby hashes

.each

accesses each element of a hash, starting from the beginning (and collecting each key and value in a separate array in its return value).

hsh = {'a' => 1, 'b' => 2, 'c' => 3}
hsh.each do |element|
print element * 2
end
# => ["a", 1]["b", 2]["c", 3]

.each_key, .each value

accesses each key OR value of a hash, starting from the beginning.

hsh = {'a' => 1, 'b' => 2, 'c' => 3}
hsh.each_key do |key|
print key * 2, ' '
end
# => aa bb cc

.each_pair

accesses each key-value pair in a hash, starting from the beginning.

hsh = {'a' => 1, 'b' => 2, 'c' => 3}
hsh.each_pair do |key, value|
puts key, ' is the key of ', value
end
# => a is the key of 1 # > b is the key of 2 # > c is the key of 3

JavaScript arrays

for loop

uses an incrementing counter which can apply to any array index

var arr = ['a', 'b', 'c', 1, 2, 3]
for (var i=0; i<6; i++)
console.log(arr[i]);
// > abc123

while loop

performs a task while a condition evaluates to true

var arr = ['a', 'b', 'c', 1, 2, 3]
var i = 0;
do {
i += 1;
console.log(arr[i]);
} while (i < 3 );
// > abc123

JavaScript objects

for..in statement

iterates over properties within an indicated object

var obj = { a: 1, b: 2, c: 3}
var props = "";
for (var property in obj) {
props += obj[property] + ", ";
}
console.log (props + "are properties of obj!");
// > 1, 2, 3, are properties of obj!

for..of statement

This statement is in ECMA 6, but does NOT have broad browser support!

iterates over values of properties within an indicated object

var obj = { a: 1, b: 2, c: 3}
var vals = "";
for (var value of obj) {
vals += obj[value] + ", ";
}
console.log (vals + "are properties of obj!");
// > 1, 2, 3, are properties of obj!
